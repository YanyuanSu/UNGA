### 原数据

所有原数据来源于[联合国官方网站](https://www.un.org/zh/ga/documents/gapv.shtml)。

原数据是以PDF的格式存在，我们采用了abbyy finereader 15进行了PDF到txt的转换，最终处理成本语料库。
